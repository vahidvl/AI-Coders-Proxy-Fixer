# 🚀 مدیریت پروکسی و رفع تحریم برای برنامه‌نویسان هوش مصنوعی 
> یک ابزار سبک و پرتابل برای دور زدن تحریم‌ها و رفع خطاهای اتصال در دستیارهای برنامه‌نویسی هوش مصنوعی مثل Claude Code، Antigravity IDE، Cursor، OpenAI Codex و VS Code.

⭐️ **اگر این ابزار مشکل شما رو حل کرد و باعث شد ساعت‌ها وقتتون هدر نره، لطفاً به این ریپازیتوری استار (STAR ⭐) بدید تا بقیه دولوپرها هم بتونن پیداش کنن!** ⭐️

## 🌍 نیازسنجی و مخاطبین هدف
این پروژه مشخصاً برای برنامه‌نویسانی توسعه داده شده که در کشورهای تحت تحریم (مثل **ایران، روسیه، کوبا، سوریه و ...**) زندگی می‌کنند، یا پشت فایروال‌های سخت‌گیرانه شرکتی قرار دارند.

دستیارهای هوش مصنوعی فوق‌العاده هستند، اما به دلیل محدودیت‌های جغرافیایی یا حساسیت‌های مربوط به پروتکل‌های TLS/SSL، معمولاً با نرم‌افزارهای عبور از تحریم (مثل v2ray، Shadowsocks و VPNها) به مشکل می‌خورند و کرش می‌کنند. اگر وسط کد زدن با ارورهایی مثل `location not supported` یا قطعی ایجنت مواجه شدید، این برنامه برای شماست.

## 🎯 خطاهایی که برای همیشه حل می‌شوند
اگر توی ردیت، فروم‌ها یا شبکه ایکس (توییتر) دنبال حل این ارورها گشتید، این برنامه همون چیزیه که نیاز دارید:

* ❌ **خطای تحریم گوگل:** `User location is not supported for the API use.`
* ❌ **خطای کلود کد:** `Post "https://cloudcode-pa.googleapis.com/v1internal:loadCodeAssist": EOF`
* ❌ **کرش کردن ایجنت:** `Agent terminated due to error. You can prompt the model to try again...`
* ❌ خطای `failed to resolve cascade config` (در Antigravity IDE)
* ❌ خطای `failed to check terminal shell support` (در Antigravity و VS Code)
* ❌ خطای `connection refused` یا `ETIMEDOUT` در ادیتورهای Cursor و Void

## ✨ ویژگی‌های کلیدی
- **تغییر وضعیت با یک کلیک:** پروکسی رو برای کل سیستم و ابزارهای برنامه‌نویسی فعال یا غیرفعال کنید.
- **ادغام با سیستم‌عامل:** تنظیم خودکار متغیرهای محیطی ویندوز (`HTTP_PROXY`, `HTTPS_PROXY` و غیره).
- **تزریق مستقیم به تنظیمات IDE:** آپدیت خودکار فایل `settings.json` در **VS Code, Antigravity IDE, Cursor و Void** با قابلیت بای‌پاس کردن `localhost` برای جلوگیری از کرش کردن سرورهای لوکال!
- **اجرا در System Tray:** برنامه در تسک‌بار مینیمایز میشه و همیشه در دسترسه.
- **کاملاً پرتابل:** بدون نیاز به نصب.

## 🛠 راهنمای نصب و استفاده
۱. **دانلود:** به صفحه [Releases](#) برید و فایل `AI_Coders_Proxy_Manager.exe` رو دانلود کنید.
۲. **اجرا:** برنامه کاملاً پرتابل است. روی آن دابل کلیک کنید.
۳. **تنظیمات:** آی‌پی و پورت پروکسی خودتون رو وارد کنید (مثلاً `127.0.0.1:10808` برای v2rayN یا Nekoray).
۴. **فعال‌سازی:** روی **Enable Proxy** کلیک کنید.
۵. **ری‌استارت ادیتور:** *حتماً یک بار IDE (مثل VS Code) یا ترمینال خودتون رو ببندید و دوباره باز کنید تا تغییرات اعمال بشه!*
